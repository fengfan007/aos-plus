## aos-plus （Animate On Scroll Plus）

> aos-plus使用javascript基于animate.css实现的一款轻量级的动画滚动库。

## aos-plus和aos库的区别

aos-plus是aos的升级版，aos-plus在aos的基础上增加了动画滚动的功能，并且支持自定义动画。
